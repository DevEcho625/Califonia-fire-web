About This Project

This project visualizes wildfire risk areas in California using a heatmap. It is built with Leaflet.js and displays recent fire data to help residents understand wildfire risks in their area.

Data Sources

  • California Department of Forestry and Fire Protection (CAL FIRE)
  • U.S. Geological Survey (USGS)

Technologies Used

  • Leaflet.js - JavaScript library for interactive maps
  • Leaflet.heat - Heatmap plugin for Leaflet
  • HTML, CSS, JavaScript
